# helper_seed

This folder mirrors the helper-side persona files needed to stand up `workspace-mc-helper`.

Use it when:
- the bridge repo is uploaded or shared without the live helper workspace
- you need a bridge-local copy of the helper persona docs

Workflow:
1. Edit the live helper workspace files first.
2. Mirror the same contents here in the same change.
3. On a fresh machine, copy these files into the helper workspace root before running the bridge.

Mirrored files:
- `AGENTS.md`
- `BOOTSTRAP.md`
- `IDENTITY.md`
- `SOUL.md`
- `USER.md`
